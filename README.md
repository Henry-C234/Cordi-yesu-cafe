# Elegant Hall - Premium Event Venue Website

A modern, elegant, and fully responsive website for a premium event hall venue. Built with HTML5, CSS3, and vanilla JavaScript, featuring smooth animations, high-quality design, and an intuitive user experience.

## 🌟 Features

### ✨ Design & User Experience
- **Modern & Elegant Design**: Clean, luxurious design with a sophisticated color scheme
- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices
- **Smooth Animations**: CSS animations and transitions for enhanced user experience
- **Professional Typography**: Beautiful fonts (Playfair Display & Inter) for elegant presentation

### 🏠 Website Sections
1. **Hero Section**: Full-width video background with compelling call-to-action
2. **About Section**: Hall information, amenities, and unique features
3. **Gallery**: Filterable photo gallery with lightbox functionality
4. **Services**: Comprehensive service offerings with icons and descriptions
5. **Testimonials**: Client reviews with carousel slider
6. **Booking System**: Complete reservation form with validation
7. **Contact**: Contact form, location map, and business information
8. **Footer**: Quick links, newsletter signup, and social media

### 🚀 Interactive Features
- **Responsive Navigation**: Sticky navbar with mobile hamburger menu
- **Gallery Filtering**: Filter events by category (weddings, conferences, parties)
- **Lightbox Gallery**: Click images to view in full-screen modal
- **Testimonials Slider**: Auto-rotating client testimonials with navigation
- **Form Validation**: Client-side validation with user feedback
- **Smooth Scrolling**: Seamless navigation between sections
- **WhatsApp Integration**: Floating WhatsApp button for quick contact
- **Scroll Animations**: Elements animate as they come into view

### 📱 Mobile-First Features
- **Touch-Friendly**: Optimized for mobile devices
- **Responsive Grid**: Flexible layouts that adapt to all screen sizes
- **Mobile Menu**: Collapsible navigation for small screens
- **Touch Gestures**: Swipe-friendly gallery and testimonials

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and modern HTML features
- **CSS3**: Advanced CSS with CSS Grid, Flexbox, and custom properties
- **JavaScript (ES6+)**: Modern JavaScript with modules and async/await
- **Font Awesome**: Professional icons for enhanced UI
- **Google Fonts**: Beautiful typography (Playfair Display & Inter)
- **Intersection Observer API**: For scroll-based animations
- **CSS Custom Properties**: For consistent theming and easy customization

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation

1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Or** set up a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

### Development Setup

1. **Customize Content**: Edit `index.html` to update text, images, and contact information
2. **Modify Styling**: Update `style.css` to change colors, fonts, and layout
3. **Add Functionality**: Enhance `script.js` with additional features
4. **Test Responsiveness**: Use browser dev tools to test different screen sizes

## 🎨 Customization

### Color Scheme
The website uses CSS custom properties for easy color customization:

```css
:root {
    --primary: #d4af37;      /* Gold */
    --secondary: #1a1a1a;    /* Dark */
    --accent: #8b4513;       /* Brown */
    --text: #333;            /* Text color */
    --light: #f8f9fa;       /* Light background */
    --white: #ffffff;        /* White */
}
```

### Typography
- **Headings**: Playfair Display (serif)
- **Body Text**: Inter (sans-serif)
