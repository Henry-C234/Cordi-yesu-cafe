# Eleganza Hall - Premium Event Venue Website

A modern, elegant, and fully responsive website for Eleganza Hall, a premier event venue specializing in weddings, conferences, parties, and special celebrations.

## 🌟 Features

### Design & User Experience
- **Modern & Elegant Design**: Luxurious gold, white, and black color scheme
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: CSS transitions and scroll-triggered animations
- **Professional Typography**: Google Fonts (Playfair Display & Inter)
- **High-Quality Images**: Curated stock photography from Unsplash

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Basic understanding of HTML, CSS, and JavaScript
- Web server (for full functionality, especially contact forms)

### Installation
1. Download or clone the project files
2. Open `index.html` in your web browser
3. For full functionality, serve the files through a web server

### File Structure
```
eleganza-hall/
│
├── index.html              # Home page
├── about.html              # About page
├── gallery.html            # Gallery page
├── services.html           # Services page
├── booking.html            # Booking page
├── testimonials.html       # Testimonials page
├── contact.html            # Contact page
├── README.md               # This file
│
├── css/
│   ├── style.css           # Main styles
│   └── responsive.css      # Responsive styles
│
└── js/
    └── main.js             # Main JavaScript functionality
```

## 🎯 Customization

### Colors & Branding
Update the CSS variables in `style.css`:
```css
:root {
    --primary-color: #d4af37;    /* Gold */
    --secondary-color: #1a1a1a;  /* Dark */
    --accent-color: #f8f9fa;     /* Light gray */
    /* ... other variables */
}
```

### Content Updates
- Replace placeholder text with your venue information
- Update contact details in all relevant files
- Replace images with your own venue photos
- Modify service offerings and pricing

### Google Maps
Replace the Google Maps embed in `contact.html` with your venue's location:
```html
<iframe src="YOUR_GOOGLE_MAPS_EMBED_URL"></iframe>
```
